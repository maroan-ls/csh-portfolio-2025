# CSH Portfolio 2025 — Maroan El Sirfy

## 1) Reconstructing Financial Networks with Valued ERGMs (MSc Thesis)
Independent, reproducible modeling on syndicated-loan–type data (institution→institution, directed/weighted). Valued ERGM recovers density, clustering, small-world geodesics, and exposure tails; observed degree/geodesics within 95% simulation envelopes; closest joint fit vs. Erdős–Rényi, Watts–Strogatz, Barabási–Albert.  
**PDF:** [PDF](/docs/MSc_Thesis.pdf) · **Code:** [https://github.com/maroan-ls/synthetic_networks](https://github.com/maroan-ls/synthetic_networks)

## 2) Syndicated-Loans ERGM Pipeline (Code)
Documented, tested pipeline (clean → fit → GOF → sims). Geometric reference with weight scale; ready for borrower-heterogeneity and temporal extensions.

## 3) Forecasting U.S. Macro Indicators (BSc Thesis)
Compared ARIMA/VAR/LSTM on GDP, PCE, Debt, UR, NUR (1970–2021). LSTM lower RMSE on GDP/UR/NUR; ARIMA best on PCE/Debt.
